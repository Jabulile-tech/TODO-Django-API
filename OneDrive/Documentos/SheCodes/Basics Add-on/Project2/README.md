# Documentary Landing Page

## Overview

This project is a landing page dedicated to the hobby of watching documentaries. It showcases different types of documentaries, highlights some favorites, and provides a visually engaging experience with modern CSS features and a dark/light theme toggle.

## Features

- **CSS Background Properties:** Uses gradient backgrounds for both light and dark themes, with a color palette of red, white, black, and brown.
- **CSS Transform Properties:** Elements such as buttons and cards use scale and rotate transforms for interactive effects.
- **CSS Grids:** Two grid galleries display documentary categories and favorite picks in a responsive layout.
- **Dark/Light Theme:** Toggle button allows switching between dark and light modes, updating background and text colors.
- **Header Image:** The header image is styled and centered to match the size of gallery images.
- **Highlight Button:** A visually prominent button links to a curated list of true crime documentaries.
- **Most Popular Section:** The most popular documentary is featured with a direct link to watch it, positioned next to the section title.
- **Quote:** Includes a popular quote among documentary lovers.
- **Footer:** Your name is displayed at the bottom of the page.

## How It Works

- All styles are written in a `<style>` block within the HTML file.
- The theme toggle button uses JavaScript to switch between dark and light modes by toggling a CSS class on the `<body>`.
- Images and content are organized using CSS Grid for a clean, responsive layout.
- Custom colors are set using CSS variables in the `:root` selector.

## File Structure

- `index.html`: Main landing page with all content and styles.
- `README.md`: This documentation file.

## Customization

- You can change images, captions, and documentary links by editing the HTML.
- To adjust colors, modify the CSS variables in the `:root` section.
- Add more documentary cards by duplicating the `.photo-card` elements in either grid gallery section.
