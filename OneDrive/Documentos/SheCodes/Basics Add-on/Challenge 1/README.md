# Johannesburg Hometown Page Challenge

## Overview

This challenge was to create a simple HTML page about my hometown, Johannesburg, South Africa. The page features a beautiful background image of the city and some descriptive text overlayed on top, styled for readability and visual appeal.

## Features

- Full-screen background image of Johannesburg
- Text overlay with a title and a short description about the city
- Responsive and modern design using CSS

## Instructions

1. Open the `index.html` file in your browser to view the page.
2. The background image is loaded from an external URL, so an internet connection is required.

## Technologies Used

- HTML5
- CSS3

## About Johannesburg

Johannesburg, also known as Jo'burg or Egoli, is the largest city in South Africa. Known for its vibrant culture, rich history, and stunning skyline, Johannesburg is the economic heart of the country. From the bustling streets of Sandton to the historic sites of Soweto, there's always something to discover in this dynamic city.
